Java Programming Guide

Java Overview

Java is a high-level, class-based, object-oriented programming language designed to have as few implementation dependencies as possible. It is widely used for building secure and robust applications.

  • Platform Independent: Java code runs on any device with a JVM (Java Virtual Machine).
  • Popular Applications: Web apps, mobile apps, enterprise applications, and more.

Java Basics

The basics of Java include understanding variables, data types, operators, and input/output operations.

public class Main {
  public static void main(String[] args) {
    int number = 10;
    String message = "Hello, Java!";
    System.out.println(message + " The number is " + number);
  }
}

Explanation:

  • int is a data type for integers.
  • String is a class representing sequences of characters.
  • System.out.println() prints a message to the console.

Java Flow Control

Flow control refers to the decision-making structures like if, switch, loops (for, while, do-while).

// Example: Switch Case
int day = 2;
switch(day) {
  case 1: System.out.println("Monday"); break;
  case 2: System.out.println("Tuesday"); break;
  default: System.out.println("Other Day");
}

Explanation:

  • switch evaluates the variable day.
  • The matching case is executed, followed by break.
  • If no cases match, the default block runs.

Java Methods

Methods are blocks of code that perform specific tasks. They are reusable and can accept parameters.

public class Calculator {
  int add(int a, int b) {
    return a + b;
  }

  public static void main(String[] args) {
    Calculator calc = new Calculator();
    System.out.println("Sum: " + calc.add(5, 10));
  }
}

Explanation:

  • add is a method that accepts two integers and returns their sum.
  • The method is invoked using an object of the class.

Java OOPs Concepts

Object-Oriented Programming in Java includes inheritance, encapsulation, abstraction, and polymorphism.

class Animal {
  void eat() {
    System.out.println("This animal eats food.");
  }
}

class Dog extends Animal {
  void bark() {
    System.out.println("The dog barks.");
  }
}

Dog dog = new Dog();
dog.eat();
dog.bark();

Explanation:

  • Inheritance allows Dog to reuse Animal's methods.
  • Additional behavior (bark) is added in the child class.

Java Interfaces

Interfaces in Java define a contract that classes must follow. They specify methods that a class must implement but do not provide implementations themselves. Interfaces are used to achieve abstraction and multiple inheritance in Java.

// Example: Java Interface
interface Animal {
void sound(); // Abstract method
}

class Dog implements Animal {
public void sound() {
  System.out.println("The dog barks.");
}
}

public class Main {
public static void main(String[] args) {
  Animal dog = new Dog();
  dog.sound();
}
}

Explanation:

  • The Animal interface defines the method sound() without implementation.
  • The Dog class implements the interface and provides its specific implementation of sound().

Java Collections

The Java Collections Framework provides data structures like lists, sets, and maps to manage dynamic and complex data efficiently. Common implementations include ArrayList, HashSet, and HashMap.

// Example: ArrayList and HashMap
import java.util.ArrayList;
import java.util.HashMap;

public class Main {
public static void main(String[] args) {
  // ArrayList Example
  ArrayList list = new ArrayList<>();
  list.add("Java");
  list.add("Python");
  list.add("C++");
  System.out.println("Languages: " + list);

  // HashMap Example
  HashMap map = new HashMap<>();
  map.put(1, "One");
  map.put(2, "Two");
  map.put(3, "Three");
  System.out.println("HashMap: " + map);
}
}

Explanation:

  • ArrayList is a resizable array. You can add, remove, or access elements dynamically.
  • HashMap stores key-value pairs, making it ideal for fast lookups.

Java Multithreading

Multithreading in Java allows multiple threads to run concurrently, enabling better utilization of CPU resources. Threads can be created by extending the Thread class or implementing the Runnable interface.

// Example: Multithreading
class MyThread extends Thread {
public void run() {
  System.out.println("Thread is running.");
}
}

public class Main {
public static void main(String[] args) {
  MyThread thread = new MyThread();
  thread.start(); // Starts a new thread
}
}

Explanation:

  • start() begins a new thread, which calls the run() method.
  • Threads execute concurrently, improving performance in certain scenarios.

Java File Handling

Java provides APIs to create, read, write, and delete files. The File, FileReader, and FileWriter classes are commonly used.

// Example: Writing to a file
import java.io.FileWriter;
import java.io.IOException;

public class Main {
public static void main(String[] args) {
  try {
    FileWriter writer = new FileWriter("output.txt");
    writer.write("Hello, Java File Handling!");
    writer.close();
    System.out.println("File written successfully.");
  } catch (IOException e) {
    System.out.println("An error occurred: " + e.getMessage());
  }
}
}

Explanation:

  • FileWriter is used to write data to a file.
  • The close() method ensures the file is properly saved and closed.

Java Regex

Regex (Regular Expressions) is used for pattern matching in strings. Java provides the Pattern and Matcher classes for this purpose.

// Example: Regex Matching
import java.util.regex.*;

public class Main {
public static void main(String[] args) {
  String text = "Welcome to Java!";
  String pattern = "Java";
  Pattern p = Pattern.compile(pattern);
  Matcher m = p.matcher(text);

  if (m.find()) {
    System.out.println("Pattern found at index: " + m.start());
  } else {
    System.out.println("Pattern not found.");
  }
}
}

Explanation:

  • Pattern.compile() compiles a regex pattern.
  • Matcher is used to search for the pattern in the text.
  • find() checks if the pattern exists, and start() returns the starting index of the match.
